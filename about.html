Sobre mim

Como iniciante no desenvolvimento web, estou empolgado em mergulhar no mundo da criação de páginas web usando HTML e CSS. Aprendendo a estrutura básica do HTML, estou descobrindo como criar elementos como títulos, parágrafos e imagens para dar forma ao conteúdo da minha página.

Além disso, estou explorando o CSS para adicionar estilo e personalização, aprendendo a definir cores, fontes e layouts para tornar minha página visualmente atraente. Embora eu esteja no início da minha jornada, estou animado para aprimorar minhas habilidades e eventualmente criar projetos web incríveis que compartilhem informações e inspirem os visitantes.
